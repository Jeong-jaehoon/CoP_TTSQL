🤖 뽑아조란?

뽑아조는 자연어를 SQL 쿼리로 변환해주는 AI Assistant입니다. 복잡한 SQL을 몰라도 일상 언어로 데이터베이스에 질문할 수 있어요!

💡 사용 방법

• 일상 언어로 질문을 입력하세요
• 뽑아조가 SQL 쿼리를 생성합니다
• 실제 데이터베이스에서 결과를 가져옵니다
• 결과를 보기 쉬운 표로 보여드려요

📊 사용 가능한 데이터

• 직원 정보: 이름, 부서, 급여, 입사일
• 부서 정보: 부서명, 관리자, 예산
• 프로젝트: 프로젝트명, 기간, 예산
• 참여 관계: 직원-프로젝트 연결 정보

✨ 질문 예시

• "급여가 가장 높은 직원 3명은?"
• "개발팀에 몇 명이 있나요?"
• "예산이 1억원 이상인 프로젝트들"
• "김철수가 참여하는 프로젝트는?"
